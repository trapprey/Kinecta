# Kinecta
Kinecta -  это проект компьютерного зрения, который превращает твою руку в игровой контроллер
Камера в реальном времени считывает 21 точку на твоей руке и строит из них светящийся скелет - как рентген. Никаких перчаток, никаких устройств, только рука перед камерой.
Когда ты сводишь большой и указательный палец, нейросеть от Google распознаёт жест и мгновенно отправляет нажатие клавиши напрямую в браузер — даже если он стоит за окном скрипта. Динозавр прыгает.
Под капотом работает MediaPipe (нейросеть от Google для трекинга рук), OpenCV для видео и отрисовки скелета, и Windows API для отправки клавиш без переключения фокуса.
Это база - по сути готовая платформа под любые жесты и любые игры или приложения
